vue2.0高仿饿了么
-----------
这个项目是使用vue2.0高仿饿了么外卖app的商家页面。开发和构建使用node.js+webpack。
项目截图：
### 商品列表页 ###
 - 这里的菜单栏和商品列表的联动效果我并没有使用better-scroll，而是用原生js写的。因为像这种溢出直接css设置scroll不就可以了嘛。点击左侧菜单栏后右侧列表栏滚动则是使用for+setTimeout实现的过度滚动，同样不需要better-scroll就可以做到。
